## Project setup

```
yarn install
```

## Install JSON Server

```
yarn global add json-server
```

## Run JSON Server

```
cd /go/to/your/project/folder
json-server db.json --watch
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Lints and fixes files

```
yarn run lint
```
